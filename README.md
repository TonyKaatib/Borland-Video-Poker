# Borland Video Poker

Much happened. Visual Studio 2026 was taking too much space on my PC drive, and I had not much choice but delete it order to gain some space. This made me go back to Embarcadero Dev C++ for the time being. Without DirectX or any API, I decided the old school way and "why not make a video game with Borland's graphic headers?"

After some days of typing and polishing the game with GPT (I'll explain why I used it later on), there you have it, my first 'official' video-game, an implementation of video-poker in the Borlands graphics header. The reason I went for GPT-ing the rest of the game other than just finishing it all by myself is honestly since I have taken a lot of C++ time with experimental game-deving and giving up what I had donme previously in Visual Studio, I decided to end this journey by taking this shortcut. Game-deving is, was fun, but it is the time to move on. I personally want to move ahead of my life and master illusionism as a hobby, for example.

So there, abra-ca-dabra. There is nothing much to say about this, you can use the mouse to select which cards you want to hold into. Press the buttons and voilá. But it works atleast.
